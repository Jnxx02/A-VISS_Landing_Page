# A-VISS Landing Page

A premium research showcase website for **A-VISS (Automated Vital Intelligence for Stress Surveillance)**.

## Overview

A-VISS is a healthcare-focused AI research project designed to support stress surveillance in clinical environments through physiological signal analysis and personalized machine learning.

This landing page presents:

* Research background
* Alarm fatigue problem
* Subject-Dependent CNN-LSTM approach
* Research methodology
* Validation results
* Dashboard showcase
* Thesis and project resources

## Research Highlights

### Dataset

* 18 Nurse Participants
* 11,509,051 Physiological Records
* 2,301,604 Time-Series Windows
* 6 Physiological Features

### Model

* Subject-Dependent CNN-LSTM
* Binary Stress Classification
* Personalized Physiological Baseline Learning

### Performance

* Accuracy: 85%
* Weighted F1 Score: 86%
* Subject-Independent Baseline: 65%
* Improvement: +20 Percentage Points
